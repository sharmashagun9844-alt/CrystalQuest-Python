#  Crystal Quest: A Logic-Driven Adventure Engine

A playable adventure game that bridges backend Python logic with a polished frontend UI.

##  Features
- **Inventory System:** Logic checks to see if you have the "Map" before progressing.
- **State Management:** Tracks player health and location dynamically.
- **Themed UI:** A dark-mode interface designed for immersion.

##  Tech Stack
- **Python:** Game engine and decision logic.
- **HTML/CSS:** Visual interface and layout.
- **Google Colab:** Initial development environment.
